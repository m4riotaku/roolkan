# Single Page Application (Ln, wn, etc) 
El objetivo de esta aplicación es compartir traducciones
de "web novels" u otros formatos de publicación(ln, manga, manwhas, etc) sin anuncios, sin suscripciones, sin venta de libros virtuales como (pdf, word, etc) de nuestras traducciones. Colaboradores que quieran unirse a nuestro equipo ir a [Contributing](./CONTRIBUTING.md) sección sugerencias y roles.

## Guia de instalacion
### Instalacion de dependencias
- [git](https://git-scm.com/install/)
- [node](https://nodejs.org/en/download/)

### Clone este repositorio
1. clone el repositorio en una carpeta de su preferencia
```
git clone https://github.com/Guchangge28/roolkan.git
```
2. entrar en la carpeta del repositorio
```
cd roolkan
```
3. instalar paquetes
```
npm install
```
4. iniciar la aplicación en modo desarrollo
```
npm run dev
```
